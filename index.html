more plates is a small web app to help you build a workout plan.


To start, click create plan in the sidebar.

Use these top tabs and choose a day


You can make each day a rest day or add exercises. Use the search bar to search.

As you add exercises, you can delete, drag to reorder, and enter in values.

On a side note, checking the rest day box will allow you to write notes that will show up in the final plan. If you want a day to be completely empty in the final plan, don't choose rest or any exercises


When done, hit view plan. In this view, you can see the whole workout in table form. You are still free to move around rows and edit values. You can also download the table as a .CSV to customize it or print.
